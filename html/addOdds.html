Functions : Add Odds Up To

sumarImparesHasta(numero)

Crear una función sumarImparesHasta que tome como argumento un número y que devuelva la suma de todos los impares empezando desde 0 hasta dicho numero inclusive.

sumarImparesHasta(5) // 9 (1 + 3 + 5 = 9)

sumarImparesHasta(13) // 49

sumarImparesHasta(47) // 576
