Functions : Get Index

ObtenerIndice(valor, array)

Crear una función obtenerIndice que tome como argumento un valor cualquiera y un array cualquiera y devuelva el índice del primer ítem con dicho valor en el array, o -1 si no hay ninguno.

obtenerIndice(12, [5, 7, 12, 34, 54, 2, 12]) // 2

obtenerIndice(83, [5, 7, 12, 34, 54, 2, 12]) // -1
